Sweat Guru is a better way to discover fitness. Self-described as the Open Table for fitness, Sweat Guru's web application allows fitness geeks to work-out on-the-go.

Agent Creative was brought in as a Designer to help develop a simple brand and scalable web application. This began with a simple road-map of user interaction, MVP goals and basic brand visuals that would be carried out through the entire experience.
